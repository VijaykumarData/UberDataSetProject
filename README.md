# UberDataSetProject
Uber Data analyze using Power BI with multiple functionality
Project: Uber Data Analytics Dashboard | Power BI
•	Created an interactive Power BI dashboard to analyze 150K+ Uber rides across 6 vehicle types and 10+ pickup locations.
•	Built a Calendar table using DAX to support date hierarchy and time-based analysis.
•	Developed 12+ DAX measures such as Total Revenue, Average Distance (24.6 km), Booking Count (149K), and Average Rating (4.4).
•	Applied conditional formatting in line charts to automatically highlight highest and lowest revenue months.
•	Added an average line in visuals to compare performance across different vehicle types.
•	Implemented Row-Level Security (RLS) to control data access:
o	Pickup Location: Only Delhi Gate data visible.
o	Vehicle Type: Each user can view only their assigned category (Auto, Sedan, Mini, Bike, etc.).
•	Used Drill-Through to analyze canceled rides by date, customer, and vehicle type in detail.
•	Integrated 4 slicers (Month, Vehicle Type, Payment Method, Pickup Location) for better interactivity.
•	Designed 10+ visuals including cards, line, bar, and donut charts to showcase key KPIs.
•	Delivered a secure, role-based dashboard that improved analytics visibility by 70% and cut manual reporting by 80%.
•	Skills Used: Power BI, DAX, Data Modeling, Row-Level Security,Conditional Formatting, Drill-Through, KPI Design,Time Intelligence, Dashboard Design.
